    Linear Regression1 #

    Regression이란 일정 데이터의 분포를 가장 잘 설명 할 수 있는 직선의 방정식을 나타냅니다.

    Regression

    \( H(X) = WX+B ( y = ax + b ) \)

    \( H : Hypothesis (가설) \)

    \( W : Weight (기울기) \)

    \( B : Bias (절편) \)

    \( Cost : H(x) - y \)

    Regression에서 Cost

    \( \frac{(H(x_1)-y_1)^2 + (H(x_2)-y_2)^2 + ... + (H(x_n)-y_n)^2 }{n} \)

    \( cost(W) = \frac{1}{n}\, \textstyle\sum_{i=1}^n \,(Wx_i-y)^2 \)

    Derivative #

    Derivative

    \( f'(x) = \frac{\Delta f(x)}{\Delta x} = \lim\limits_{\Delta x \rightarrow 0 } \, \frac{f(x+\Delta x)-f(x)}{\Delta x} \)

    Derivative Basic

    \( f(x) = constant -> f'(x) = 0 \)

    \( f'(x) = e ^ x => f'(x) = e ^ x \)

    \( f'(x) = e ^ -x => f'(x) = -e ^ -x \)

    \( f'(x) = ax ^ x => f'(x) = nax ^ n-1 \)

    \( f'(x) = ln ^ x => f'(x) = {1} /over {x} \)

    \( \frac{1}{x} = x ^ -1 \)

    Partial Derivative

    \( f(x,y) = 2x + 3xy + y^3 \)

    \( \frac{\partial f(x,y)}{\partial x} = \frac{\partial(2x+3xy+y^3)}{\partial x} = 2 + 3y \)

    Chain Rule

    \( \frac{\partial f}{\partial x} = \frac{\partial f}{\partial t} \circ \frac{\partial t}{\partial x} \)

    \( f(x) = e ^ 3x^2 : e^t , t = 3x^2 \)

    \( \frac{\partial f(x,y)}{\partial x} = \frac{\partial(2x+3xy+y^3)}{\partial x} = 2 + 3y \)

    Numerical Derivative

    \( f(x) = x^2 \)

    \( f(x) = 3xe^x \)

    Loss Function

    \( Loss Funtion = \frac{(t_1-y_1)^2 + (t_2-y_2)^2 + ... + (t_n-y_n)^2}{n} \)

    \( = \frac{[t_1-(Wx_1+b)]^2 + [t_2-(Wx_2+b)]^2 + ... + [t_n-(Wx_n+b)]^2}{n} \)

    \( = \frac{1}{n} \, \displaystyle\sum_{i=1}^n \, [t_i-(Wx_i+b)]^2 \)

    Learning Rate

    \( W = W - \alpha \, \frac{\partial E(W,b)}{\partial W} \)

    Sigmoid

    \( Z = Wx + b \)

    \( y = sigmoid(Z) = \sigma (z) = \frac{1}{1+e^-z} \)

    Cross Entropy

    \( y = \frac{1}{1+e^-(Wx+b)} , t_i = 0 or 1 \)

    \( E(W,b) = - \, \displaystyle\sum_{i=1}^n \, { \lbrace t_i \, \log y_i + (1-t_i) \log (1-y_i) \rbrace } \)
